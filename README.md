# Calculadora HTML

Projeto simples de uma calculadora em HTML que permite ao usuário inserir dois números e escolher entre as operações de adição e subtração.

## Funcionalidades

- Dois campos de entrada para números
- Seleção de operação: adição ou subtração
- Cálculo automático ou via botão ao enviar os valores
- Resultado exibido na própria página

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Insira o primeiro número.
3. Insira o segundo número.
4. Escolha a operação desejada:
   - `+` para adição
   - `-` para subtração
5. Clique no botão para calcular.
6. Veja o resultado exibido na página.

## Estrutura esperada do projeto

- `index.html` - página principal com os campos de entrada, seleção de operação e botão de cálculo.
- `styles.css` - estilos visuais da página (opcional).
- `script.js` - lógica de cálculo e manipulação dos valores.

## Observações

Este projeto é ideal para aprendizado de HTML, CSS e JavaScript básicos, além de ser um bom exercício de manipulação de formulários e eventos em página web.
